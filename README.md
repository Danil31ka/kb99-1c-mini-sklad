# Мини-Склад (1С)

Учебная конфигурация 1С для учета товаров на складе.

## Реализовано

### Справочники

* Товары

* Контрагенты

### Документы

* Поступление

* Реализация

### Пoдсистема

* Mини-Склад

## Открытие проекта

1. Открыть конфигурацию в режиме Конфигуратор.

2. Выполнить загрузку конфигурации из файлов.

3. Обновить конфигурацию базы данных.

4. Запустить режим Предприятие.
